# Sales Data Analysis Project

A Python project that performs sales data cleaning, analysis, KPI generation, and reporting using Pandas.

## Features

- Generates synthetic sales datasets
- Loads custom CSV files for analysis
- Cleans duplicates, null values, and outliers
- Calculates KPIs such as revenue, average order value, and units sold
- Analyzes performance by product, region, category, sales rep, and channel
- Generates monthly trends and growth percentages
- Saves cleaned datasets and summary reports to CSV files

## Technologies Used

- Python
- Pandas
- NumPy
- argparse

## Installation

pip install pandas numpy

python task3_data_analysis.py
